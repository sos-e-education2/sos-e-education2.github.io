I-3  Solution saturée-solution non saturée- solubilité

Exemple :

On dissout du sel dans de l’eau, on agite pour rendre le mélange homogène.

Ceci est possible que si la quantité de sel est petite.

A partir d’une certaine quantité, le sel ne peut plus se dissoudre.  A ce moment, on dit alors que la solution est saturée.

✓ Solution saturée :

Une solution est dite saturée lorsque son solvant ne peut plus dissoudre du soluté.

✓ Solution non saturée :

Une solution est dite non saturée quand son solvant peut dissoudre du soluté.

✓ Solubilité

La solubilité d’une substance dans un solvant est la quantité maximale de matière de cette substance que l’on peut dissoudre dans un volume donné de solvant.

Elle varie avec la température :

Exemple :
