II.3. Concentration molaire volumique ou molarité

La concentration molaire volumique C d’une solution est la quantité de matière n de soluté dissout par litre (V) de solution. Elle s’exprime en 𝑚𝑜𝑙.𝐿 −1.

C : concentration molaire en mol.L-1

n : quantité de matière de soluté en mol

V : volume de solution en L

Remarque :

  • La concentration molaire d’un ion dans une solution, sera notée entre crochets [ ]. Par exemple on notera :

- C(NaCl) la concentration d’une solution d’eau salée

-[Na+ ] la concentration de l’ion sodium dans cette solution

--[Cl-] la concentration de l’ion chlorure dans cette solution

  • La concentration molaire d'une solution est aussi appelée la molarité.

    Exemples :

    - une solution molaire ou de molarité 1 M est une solution de concentration 1 mol/L :1M = 1 mol.L-1

    -une solution est dite décimolaire si sa concentration est C = 0,1 𝑚𝑜𝑙. 𝐿 -1

    -une solution est dite centimolaire si sa concentration est C = 0,01 𝑚𝑜𝑙. 𝐿−1
